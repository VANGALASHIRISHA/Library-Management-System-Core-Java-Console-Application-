A backend console-based Library Management System developed using Core Java and Java Collections Framework. This application enables efficient management of library records with CRUD-like operations, demonstrating strong understanding of OOP principles and backend logic implementation.

📌 Project Overview

This project is designed to manage book records in a library environment using a menu-driven console interface. It allows users to add, view, search, issue, return, and delete books.

All data is stored dynamically using Java’s ArrayList during runtime, showcasing practical implementation of Java fundamentals without frontend or database integration.

🚀 Features

✅ Add new books to the system
✅ View all available books
✅ Search books by ID or title
✅ Issue books to users
✅ Return issued books
✅ Remove books from records
✅ Menu-driven console interface
✅ Dynamic data handling using ArrayList

⚙️ How It Works

User selects options from a menu-driven console interface

Book data is stored using ArrayList

Linear Search is used to find records

CRUD operations are performed through methods

Output is formatted using the toString() method
